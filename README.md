# Student Performance Prediction (Machine Learning)
This Project predicts student academic performance using the xAPI-Edu-Data dataset.

# Dataset
xAPI-Edu-Data (480 samples, 17 features)
Features include:
- gender
- nationality
- raisedhands
- VisITedResources
- Discussion
- StudentAbsenceDays,etc.

# Machine Learning Models
- Random Forest
- Logistis Regression
- KNN

# Results
Best model : Random Forest
Accuracy : 0.83

- Python
- Pandas
- Scikit-learn
